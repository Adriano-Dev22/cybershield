## [0.1.0] - 2026-04-26
### Added
- Initial release of CyberShield SIEM
- User authentication with JWT and bcrypt
- Interactive dashboard with Chart.js graphs
- Automatic brute force detection engine
- Fake log generation for testing
- Log upload support
- Real-time alerts system
- MITRE ATT&CK threat mapping
- Full Swagger API documentation
- Docker and docker-compose support
- Jinja2 templates with Tailwind CSS + DaisyUI
- HTMX for interactivity without heavy JavaScript
